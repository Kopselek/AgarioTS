# Agario.ts

### Description

Agario.ts is reconstrucion of [Agar.io](https://agar.io/) made in typescript.
Project was made only for education purpose!

### Instalation

Download Project


Use the node package manager [npm](https://www.npmjs.com/) to install project.
```bash
npm install
```

Run project by
```
npm start
```
